Thank you for your interest in contributing to Lend-a-Read!
This project aims to help communities run their own lightweight book-sharing systems.
We welcome contributions of all kinds - code, design, documentation, localization, and security reviews.

# Ways to Contribute

## Report Bugs

Search existing issues first

Create a new issue if not found

Include steps to reproduce

Include logs/screenshots

## Suggest Features

Open an issue with label enhancement

### Describe:
  
  Problem
  
  Proposed solution
  
  Use case

## Code Contributions

Pick an issue

Comment to get assigned

Fork → Branch → PR
